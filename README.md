# 《玲珑女》EPUB 版电子书

本项目是小说《玲珑女》的 EPUB 电子书版本，**该项目为实验性质**，探索和测试如何通过 Pandoc+Markdown 生成**符合基础标准**的 EPUB 电子书。

## 小说简介

故事发生在清末民初中国江南地区一个制扇名镇。镇里出产的团扇扇面上的美人，都是依照真人的容貌绘成。小镇每隔三年便要进行一次美人“照影会”，选出新的美女。有意思的是评选美女的方式并不是靠人，而是靠鱼唇，有幸当选为扇面美人的女子均必须有“沉鱼”之貌。

故事起始的这一年，现任美人一位得了肺痨、一位跳河，还有一位也有许多流言。按照族规，镇里要补选扇面美人。镇长白立斋为了完成十几年前妻子的遗愿，命女儿——杭州女子师范学校学生白凤衣回玲珑镇参加选美。新丧双亲的孤女刘玉指，为了病重的奶奶留下救命钱，不顾男友画师袁小照的劝阻，执意要成为扇面美人。留学国外归来的秋氏族长秋莲篷之侄秋洗月，携法国籍妻子柳诗回到玲珑镇，正好遇上玲珑镇的选美会。漂亮、洋气的柳诗因为好奇也参与到选美之中，结果与白凤衣、刘玉指一起入选。

按族规祖制，中选的美人不能再有儿女私情，倘若违反，即被打入“无影小楼”，终生与世隔绝。受西方文明熏陶的秋洗月为了拯救妻子与众位扇面美人，联合大扇商魏锦人，与传统势力展开了一系列惊心动魄的斗争。在秋洗月与白凤衣等人的努力下，终于打破了封建家族的禁锢，并揭穿了大奸似忠的魏锦人的阴谋。

## 如何生成

安装 `Pandoc`、`Python`，且安装全部项目依赖。之后执行：

```bash
poe makebook # 或 uv run poe makebook（如果通过 uv 管理项目）
```

生成的 EPUB 电子书位于 `out` 目录下。

## 兼容性问题

> [!TIP]
>
> **本项目在 Readest 和 Calibre 上测试通过，且仅推荐使用这两款阅读器以获得最佳显示效果**。

由于**并非所有电子书阅读器都遵守或实现了标准且完整的 EPUB 规范**，部分阅读器可能无法准确显示本电子书，部分效果或样式可能有所不同或缺失。这是正常现象。

**例如**：

- **“静读天下”**：由于强行覆盖 `<hr>` 元素为默认样式（一条细横线），章节段落分隔符样式缺失。

## 其它信息

**原著作者**：高锋

**出版时间**：2004 年（另有同名电视剧，于 2003 年左右上映）

个人不建议购买原著小说，**实体版本内包含大量（甚至是很明显到能一眼认出的）错别字、语法错误、排版格式错误、上下文错误，似乎完全没有校对**。虽在 OCR 时尝试校对了全文并改正了上述错误，但**仍可能有余下遗漏的错误**，敬请谅解。感兴趣者可通过“当当网”或“孔夫子旧书网”购买实体版原著/剧本，或电视剧盒装发行 VCD/DVD。
